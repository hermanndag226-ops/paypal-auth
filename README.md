# PayPal Auth App

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/hermanndag226-ops/paypal-auth)

Multilingual PayPal-style authentication app with two-step login.

## Features
- Email/Password authentication
- Multilingual support (EN/FR/ES/ZH)
- Password reset
- Admin settings panel
- Persistent credential storage
